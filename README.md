# z-lux PARFUMERIE

Frontend for the z-lux perfume brand (desktop, scrollable). Built to match the Figma design.

## Folder structure

```
zlux-parfumerie/
├── index.html          # Entry page (header, hero, products, footer)
├── README.md           # This file
├── css/
│   └── styles.css      # All styles (tokens, layout, components)
├── js/
│   └── main.js         # Smooth scroll and future interactivity
└── assets/
    └── images/         # Put product images and icons here
```

## How to run

Open `index.html` in a browser, or serve the folder with any static server.

## Design

- **Colors:** `#121212`, `#1a1a1a`, `#e0c9a6`, `#f5f5f5`
- **Fonts:** Cormorant Garamond (headings), Montserrat (body)
- **Layout:** Desktop min-width 1280px, vertical scroll
